# Ta vivo - Health Check


# Development

For install all apps dependencies use;

```bash
$ yarn
```

Now you can continue.

## UI

Run development server

```bash
$ yarn dev:ui
```

## API

Create the `.env` file into `api` directory, use the `.env.example` file;

```bash
$ cp ./api/.env.example ./api/.env
```

Run development server

```bash
$ yarn dev:api
```
